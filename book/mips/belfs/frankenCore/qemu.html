14.2. Binclock

14.2.1. binclock Contents

A binary display for current time.

14.2.2. Installation of binclock

Prepare binclock for compilation, by removing all references to strip from the Makefile: 



        ./configure --help
        
        ./configure --target-list=i386-linux-user --static

Compile the package: 

make

Install the binaries, documentation, and shared libraries: 

make install
