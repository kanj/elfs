1.3. Master Changelog

This is version GIT-0.0.1-20130730 of the Cross-Compiled Linux From Scratch book, dated July 30, 2013. If this book is more than six months old, a newer and better version is probably already available. To find out, please check one of the mirrors via http://trac.cross-lfs.org/.

Below is a list of detailed changes made since the previous release of the book.
