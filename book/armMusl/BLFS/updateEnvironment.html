14.2. Update .bashrc

Adding the following to the .bashrc file will substatntially increase the number of BLFS packages that work "out of the Box". 


echo export host_alias=\""${CLFS_TARGET}\"" >> ~/.bashrc
echo export DESTDIR=\""${CLFS}\"" >> ~/.bashrc
source ~/.bashrc
