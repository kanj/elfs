13.5. Beyond the Book

The repository allows for even more possibilities: automatically building the system described in a book, adding packages to a build, and adding your own book.

13.5.1. Automatic Build

The same makefile that generates the book can also be used to automatically build a system. It's as easy as this: $ git clone git://github.com/kanj/elfs $ cd BOOK $ make armMusl-build Over the next several hours a complete system will be built in the ../sysBuild/clfs_armMusl

13.5.2. Add a Package

Starting from the main directory adding a package is as easy as: $ cd BELFS $ make ARCH=armMusl slang-add See appendix for status of supported packages.
