3.4. Needed Patches

In addition to the packages, several patches are also required. These patches correct any mistakes in the packages that should be fixed by the maintainer. The patches also make small modifications to make the packages easier to work with. The following patches will be needed to build a CLFS system:

Busybox Config Patch - 26 KB:

Download: http://patches.cross-lfs.org/embedded-dev/busybox-1.18.4-config-1.patch

MD5 sum: bd06ac3799960df8c72f73bb4ad27e30

Iana-Etc Update Patch - 864 KB:

Download: http://patches.cross-lfs.org/embedded-dev/iana-etc-2.30-update-1.patch

MD5 sum: 9b4d102672fc8bf777bbb95d1007edda

uClibc Configs Patch - 52 KB:

Download: http://patches.cross-lfs.org/embedded-dev/uClibc-0.9.31-configs-2.patch

MD5 sum: e06455fe909d2572f7926f48cd066785

Total size of these patches: about 942 KB
