2.2. Creating the Working Directory

A dedicated place to build the CLFS system needs to be created. For the purposes of this book, it is assumed that the working directory is /opt/clfs, but the directory choice is up to you.

Choose a working directory and assign it to the CLFS environment variable by running: 

export CLFS=/opt/clfs

Next, create the working directory by running: 

mkdir -pv ${CLFS}

Now that there is an established place to work, it is time to download the packages.
